# xG — Expected Goals

**Category:** Football
**Created:** 2026-04-19
**Updated:** 2026-04-19

Метрика якості атаки/захисту команди без шуму реальних голів. Один з ключових features у моделях Capper.

## Key Facts

- xG показує скільки голів мала б забити команда виходячи з якості моментів
- Прибирає variance реальних голів (щастя/невдача воротаря)
- Краще відображає реальну силу команди ніж голи
- Використовується як feature у [Capper](capper-overview.md)

## Details

Кожен удар по воротах має свою ймовірність гола (0.0–1.0) виходячи з позиції, кута, відстані, типу удару тощо. Сума xG за матч — це "справедливий" рахунок.

Команда може виграти 3:0 маючи xG 0.8 vs 2.1 — значить вони переграли суперника за якістю, але програли за xG. Наступного матчу результат скоріше за все вирівняється.

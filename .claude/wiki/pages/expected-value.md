# Expected Value (EV)

**Category:** Betting & EV
**Created:** 2026-04-19
**Updated:** 2026-04-19

Математичне очікування прибутку зі ставки. Основний фільтр у Capper — відправляються тільки ставки з позитивним EV.

## Key Facts

- Формула: `EV = (our_probability × odds) - 1`
- EV > 0 → ставка value, відправляємо
- EV < 0 → ставка програшна на дистанції, ігноруємо
- Вимагає откаліброваних ймовірностей (raw model output не підходить)

## Details

EV показує: якщо поставити цю ставку нескінченну кількість разів, скільки в середньому заробиш на одиницю ставки.

Приклад: наша модель дає P(перемога) = 0.55, букмекер дає коефіцієнт 2.10.
`EV = 0.55 × 2.10 - 1 = 1.155 - 1 = +0.155` → +15.5% від ставки на дистанції.

Якщо P = 0.45 при тих самих коефіцієнтах:
`EV = 0.45 × 2.10 - 1 = -0.055` → збиткова ставка.

Калібрація ймовірностей критична: завищені ймовірності дають хибний позитивний EV.
